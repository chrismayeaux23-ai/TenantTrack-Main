# TenantTrack 90-Day Customer Acquisition Playbook

This folder is the operations manual for getting TenantTrack from $0 → $1.5–3k MRR in 90 days through direct outreach to landlords with 10–50 units. Everything in here is a working document — open it, edit it, use it.

## Contents

| File | What it is | When to use it |
|------|-----------|----------------|
| `lead-sourcing-guide.md` | Where to find 10–50 unit landlords and how to qualify them | Week 1, then weekly top-ups |
| `lead-list-template.csv` | Spreadsheet template (open in Sheets/Excel) — column headers + 5 example rows | Copy to Google Sheets and fill it in |
| `loom-demo-script.md` | 90-second screen-recording script for the product demo | One-time recording, then re-record only if templates change |
| `outreach-templates.md` | DM, cold email, and forum-comment-reply templates with personalization slots | Every outreach send |
| `onboarding-sop.md` | Step-by-step founder playbook for every trial signup | The moment a trial signup hits your inbox |
| `tracking-system-guide.md` | How to set up and use the Google Sheet that runs the playbook | Day 1 setup, then daily updates |
| `daily-checklist.md` | The 30-minute daily routine that drives the whole thing | Every weekday morning |

## How to start (Day 1)

1. Read `tracking-system-guide.md` and create the Google Sheet (15 min).
2. Read `lead-sourcing-guide.md` and spend 4–6 hours building your first 200 leads. Paste them into the sheet.
3. Record the Loom video using `loom-demo-script.md`. Don't make it perfect — make it done.
4. Read `outreach-templates.md`. Pick one template. Send 10 personalized outreaches.
5. From Day 2 onward: open `daily-checklist.md` every morning and do the 30 minutes.

## What "done" looks like at Day 90

- 200 leads contacted, 60–100 contacted twice (one follow-up)
- 15–30 trial signups
- 10–20 onboarding calls held
- 15–25 paying customers ($50–$150/mo each)
- $1.5k–3k MRR
- 3–5 testimonials and 2–4 customer-sourced referrals
- A clear answer to: which channel + which message converts best, so you can double down for months 4–6

## North-star metric

**Replies per week.** Everything upstream and downstream of that follows. If replies dry up, change the message before you change the volume.
